🔗 Connect with Your Friend

This guide helps you set up Firebase so you and your friend can share a challenge and see each other's progress in real time.

How It Works

One person creates the challenge, gets a Share Code, and sends it to the other. Both phones sync through Firebase — when one person checks a task, the other sees it instantly.

Setup (One-Time, 5 Minutes)

  1. Go to Firebase Console and create a new project (free tier)
  2. Click Build → Realtime Database → Create Database
  3. Choose any location, then select "Start in test mode"
  4. Click Project Settings (gear icon) → scroll to "Your apps" → click the Web icon (</>)
  5. Register an app name (anything), then copy the firebaseConfig object
  6. Open firebase-config.js in your project and paste your config values

⚠️ Keep "test mode" rules for now (they expire in 30 days). For longer use, update your database rules to:

{
  "rules": {
    "challenges": {
      "$challengeId": {
        ".read": true,
        ".write": true
      }
    }
  }
}
